Tool Reference

Network
- port_scanner.py — scan target for open TCP ports
- subnet_enum.py — threaded TCP port scan and optional subdomain enumeration
- vuln_scanner.py — port scan + basic HTTP header and TLS checks
- packet_sniffer.py — basic IP/TCP/UDP/ICMP sniffing (requires sudo)

Web
- sql_injector.py — common SQLi payload reflection/error checks
- xss_tester.py — reflected XSS checks using common payloads
- web_recon.py — DNS, WHOIS, headers, SSL/TLS info, subdomains

Wireless
- wifi_deauth.py — send 802.11 deauth frames (Linux monitor mode)
- bluetooth_scanner.py — discover nearby Bluetooth devices

Exploitation
- shellcode_encoder.py — encode/decode hex/base64
- buffer_overflow.py — demonstration of out-of-bounds write detection
- keylogger.py — logs keystrokes to a file (authorized use only)

Automation
- AutoExploit.py — detect simple SQLi/XSS reflections and record PoCs
- AutoReport.py — generate a PDF report from PoC findings
