## wifi_deauth.py

### Prerequisites:
- `pip install scapy`
- `aircrack-ng` suite (for monitor mode setup)

### Usage:
- `sudo python wifi_deauth.py <Target MAC> <Gateway MAC> <Interface>`

### Example:
sudo python wifi_deauth.py 00:14:22:01:23:45 00:11:22:33:44:55 wlan0mon
